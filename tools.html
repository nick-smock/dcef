DCEF / Tools

Open-Source
Toolchain

Six Python scripts implementing the full DCEF pipeline — from raw ULog binary to standardized evaluation outputs. Real code, real outputs, built to run on your data.

Overview

Built to Run on Real Data

Each script is standalone, reads and writes standard DCEF-schema CSV files, and produces outputs that are directly comparable across sites. Install the dependencies once and run the pipeline end-to-end in under five minutes.

The synthetic Scenario Alpha dataset on the Data page was generated to match exactly what real pipeline outputs look like — so you can run every script against those CSVs right now to verify the pipeline before using real flight data.

TOOL 01
Parse ULog Telemetry
Reads a PX4 ULog binary and extracts position, heading, ground speed, and RF link quality. Resamples to 1 Hz and writes a DCEF telemetry CSV.
IN.ulg (PX4 binary)
OUTtelemetry.csv
pyulogpandasnumpy
TOOL 02
Correlate Telemetry + Sensor
Merges telemetry and sensor CSVs on timestamp_utc using pandas merge_asof. Computes range, position error, and all derived fields.
INtelemetry.csv + sensor.csv
OUTcorrelated.csv
pandasnumpy
TOOL 03
Plot Position Error Scatter
Generates a 2D horizontal position error scatter plot. Points are colour-coded by range band with concentric CEP rings and a crosshair at truth origin.
INcorrelated.csv
OUTscatter.png
matplotlibnumpy
TOOL 04
Plot Polar Directional Bias
Generates a polar rose plot showing mean position error binned by vehicle heading. Elongation along any axis indicates systematic sensor pointing bias.
INcorrelated.csv
OUTpolar.png
matplotlibnumpy

Plot PoD Curve
