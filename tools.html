Toolchain

Six Python scripts implementing the full DCEF pipeline — from raw ULog binary to standardized evaluation outputs. Real code, real outputs, built to run on your data.

Overview

Built to Run on Real Data

Each script is standalone, reads and writes standard DCEF-schema CSV files, and produces outputs directly comparable across sites. Install the dependencies once and run the pipeline end-to-end in under five minutes.

The synthetic Scenario Alpha dataset on the Data page was built to match exactly what real pipeline outputs look like — run every script against those CSVs right now to verify the pipeline before using real flight data.

TOOL 01
Parse ULog Telemetry
Reads a PX4 ULog binary and extracts position, heading, ground speed, and RF link quality. Resamples to 1 Hz and writes a DCEF telemetry CSV.
IN  .ulg (PX4 binary)
OUTtelemetry.csv
pyulogpandasnumpy
TOOL 02
Correlate Telemetry + Sensor
Merges telemetry and sensor CSVs on timestamp_utc using pandas merge_asof with 50ms tolerance. Computes range, position error, and all derived fields.
IN  telemetry.csv + sensor.csv
OUTcorrelated.csv
pandasnumpy
