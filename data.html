About the Data

Built to Be Replicated

These datasets are synthetic — generated from first principles using realistic C-UAS evaluation parameters, not derived from any operational test record. Every value is constructed to behave the way real data would: position error grows with range, detection probability degrades at far range, and directional bias appears where terrain masking would realistically occur.

The data is built around Scenario Alpha: a notional Tier 2 radar evaluation at a fixed range facility, covering approach, hover, and orbit flight profiles between 300m and 600m slant range. The correlated dataset is the same source used to generate all four research visualizations — the scatter plot, polar plot, PoD curve, and track overlay are all derived from these 162 records.

The schema template CSV gives any evaluation team a clean starting point — load it, fill it with real data from their own test run, and the DCEF analysis pipeline accepts it without modification.

Usage

How to Use the Data

Both code blocks below work out of the box with the downloaded CSV files and a standard Python environment. The DCEF Tools page provides the full analysis pipeline — these snippets are the entry point.

Python Load & inspect the correlated dataset 
import pandas as pd

# Load the full correlated dataset
df = pd.read_csv(
    "scenario_alpha_correlated.csv",
    parse_dates=["timestamp_utc"]
)

# Quick summary
print(df[[
    "range_m", "pos_error_m",
    "detection_flag"
]].describe())

# Detection rate overall
pod = df["detection_flag"].mean()
print(f"Overall PoD: {pod:.1%}")

# Mean error by range bin
df["range_bin"] = pd.cut(
    df["range_m"],
    bins=[300,350,400,450,500,550,600]
)
print(df.groupby("range_bin")[
    "pos_error_m"
].mean().round(2))
Python Filter by tier & check RF events 
import pandas as pd

df = pd.read_csv(
    "scenario_alpha_correlated.csv",
    parse_dates=["timestamp_utc"]
)

# Filter to Tier 2 records only
t2 = df[df["tier"] == 2].copy()

# Isolate any RF interference events
rf_events = df[
    df["rf_packet_loss_pct"] > 0
][[
    "timestamp_utc",
    "rf_packet_loss_pct",
    "range_m"
]]
print(f"RF events: {len(rf_events)}")

# PoD at each range bin
pod_curve = t2.groupby(
    pd.cut(t2["range_m"],
    bins=[300,350,400,450,
          500,550,600])
)["detection_flag"].mean()
print(pod_curve.round(3))
